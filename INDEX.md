## 👨‍👩‍👧‍👦 Sharing with Family

### 3 Options

1. **Local Network** (Easiest)
   - Start app on your computer
   - Family accesses via IP on same WiFi
   - Real-time sync
   - See **FAMILY_SHARING_GUIDE.md** → Option 1

2. **Export/Import** (Simple)
   - Export data as JSON
   - Share file via email
   - Family imports to their app
   - See **FAMILY_SHARING_GUIDE.md** → Option 2

3. **Cloud** (Always Available)
   - Deploy to Netlify/Vercel
   - Share public URL
   - Access from anywhere
   - See **FAMILY_SHARING_GUIDE.md** → Option 3
